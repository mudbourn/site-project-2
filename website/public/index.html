can be done. Here is a link to the Discord server.

Client Setup Guide

Download the client zip file from the
Google Drive folder.

Download and install ATLauncher from the
official website.

Open ATLauncher.

Click "Instances" on the right hand side.

At the top left of the screen, click import.

Select browse, then navigate to where the
downloaded zip file is, and import it.

It is recommended to change the name to
"MMS-AT-" immediately followed by the
version of the zip file you downloaded
(for example:"MMS-AT-2.1").

Once the import is complete, click on the
Accounts tab and log into your Minecraft
account if you haven't already done so.

Run the instance by pressing play and add
the server's address in the "Mutiplayer"
menu if it isn’t already added
(mentioned above).
